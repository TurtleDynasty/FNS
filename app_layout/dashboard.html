Overview

Select a visualization.
Sunburst: The layered locations of different domains within Spectrum Protect. Specifically the different servers, domains, and nodes that are currently active. This includes the current size of each of the servers.
Replicated Objects: The total amount of objects being replicated in a single node that is currently in use.
AF Segments: The total number of split segments located within in each of the volumes currently in use.
Deduplication Backups: The total amount of deduplication objects being backed up on currently active nodes.
Storage Pool Objects: The total amount of objects in each storage pool that is currently in use.
Storage Pool Space: The current size of each storage pool that is currently in use.
Container References: The current number of references in each container that is currently in use.
Scatter Test: This is the Scatter Plot Test description.
File Occupancy: The number of files in each storage pool that is currently in use.
